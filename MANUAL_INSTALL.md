# Ручная установка клиента светофора на Windows

## Что нужно скопировать на каждый комп:

1. **svetooofor-client.jar** - само приложение
2. **client.properties** - конфигурация (настроен на сервер 10.10.90.170:52521)

## Пошаговая инструкция установки:

### 1. Создайте папку для приложения
```
C:\Svetofor\
```

### 2. Скопируйте файлы
- Положите `svetooofor-client.jar` в `C:\Svetofor\`
- Положите `client.properties` в `C:\Svetofor\`

### 3. Создайте ярлык для запуска

**Правой кнопкой на Рабочем столе → Создать → Ярлык**

В поле "Расположение объекта" введите:
```
javaw -jar "C:\Svetofor\svetooofor-client.jar"
```

Название: `Светофор JIRA`

### 4. Добавьте в автозагрузку

Скопируйте созданный ярлык в папку автозагрузки:
```
C:\Users\<ИМЯ_ПОЛЬЗОВАТЕЛЯ>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
```

Или нажмите `Win + R`, введите `shell:startup` и вставьте туда ярлык.

### 5. Первый запуск

Запустите через ярлык - появится окно с выбором роли:
- **Пользователь** - для обычных сотрудников (только просмотр светофора)
- **Администратор** - для тех кому нужно управлять третьим кружком (очередь)

## Про иконку

Иконка встроена в jar файл, появится автоматически на панели задач и в окне приложения.

## Про пароль администратора

Для роли "Администратор" нужен пароль. Есть 2 варианта:

### Вариант 1: Без пароля (для всех пользователей)
Отредактируйте `C:\Svetofor\client.properties`, раскомментируйте строки:
```properties
admin.login=admin
admin.password=qweasd123#$
```

### Вариант 2: Переменные окружения (безопаснее)
Создайте переменные окружения Windows:
1. `Win + R` → `sysdm.cpl` → "Дополнительно" → "Переменные среды"
2. Создать переменную: `ADMIN_PASSWORD` = `qweasd123#$`

## Быстрый способ для 400 компов

### Через сетевую папку:

1. Создайте сетевую папку на сервере:
   ```
   \\10.10.90.170\svetofor
   ```

2. Положите туда `svetooofor-client.jar` и `client.properties`

3. На каждом компе создайте ярлык на:
   ```
   javaw -jar "\\10.10.90.170\svetofor\svetooofor-client.jar"
   ```

4. Ярлык в автозагрузку

**Плюс:** Обновление в одном месте → все компы автоматически получают новую версию!

## Требования

- Java 17 или выше должна быть установлена на компе
- Проверить: `java -version` в командной строке

## Проверка работы

После установки:
1. Запустите приложение
2. Выберите роль (Пользователь/Администратор)
3. В правом нижнем углу появится светофор (3 темных кружка)
4. При создании инцидента в JIRA - красный кружок загорится красным

## Если что-то не работает

- Проверьте что сервер запущен: `10.10.90.170:52521`
- Проверьте что Java установлена
- Посмотрите логи в консоли (если запустить через `java -jar` вместо `javaw`)
